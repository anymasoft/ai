# search_outscraper.py

Поиск компаний через Outscraper (Google Maps API). Production-ready скрипт с надёжной обработкой ошибок.

## Установка

```bash
# Зависимости
pip install outscraper python-dotenv requests

# Или все сразу
pip install outscraper python-dotenv requests openai
```

## Настройка .env

1. Скопировать `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Заполнить ключи:
   ```
   OUTSCRAPER_API_KEY=your_key_here
   OPENAI_API_KEY=your_key_here
   OPENAI_MODEL=gpt-4o-mini
   ```

   - Получить `OUTSCRAPER_API_KEY` на https://outscraper.com
   - Получить `OPENAI_API_KEY` на https://platform.openai.com

## Использование

### 1. Создать query.txt

```bash
echo "тату-салоны в братске" > query.txt
```

### 2. Запустить скрипт

```bash
python search_outscraper.py
```

### 3. Результаты

Результаты сохраняются в `results.csv`:

```csv
name,site
ООО "Салон красоты",http://salon.ru
Тату-мастерская "Ink",https://inktattoo.com
```

## Обработка ошибок

Скрипт корректно обрабатывает:
- ✓ Отсутствие файла `query.txt`
- ✓ Отсутствие переменных окружения
- ✓ Ошибки API Outscraper
- ✓ Ошибки OpenAI API
- ✓ Пустые результаты
- ✓ Невалидные JSON ответы

При критической ошибке скрипт выводит понятное сообщение и завершается с `exit(1)`.

## Формат query.txt

Произвольный текстовый запрос, например:
- `кофейни в москве`
- `парикмахерские в санкт-петербурге`
- `врачи-кардиологи`
- `автомойки в иркутске`

LLM автоматически распарсит:
1. **Нишу** (категория/тип бизнеса)
2. **Город** (место)

## Обработка результатов

### Структура CSV

- `name` — название компании
- `site` — веб-сайт (если доступен, иначе пусто)

### Дополнительно в API

Outscraper возвращает также:
- rating
- review_count
- phone
- type
- address
- и другие поля

При необходимости можно расширить `results.append()` для сохранения дополнительных полей.

## Лимиты

- **Outscraper:** По умолчанию `limit=100` результатов
- **Outscraper API:** Зависит от тарифа вашего аккаунта
- **OpenAI API:** Зависит от выбранной модели

Чтобы изменить лимит, отредактируйте строку в скрипте:
```python
limit=100  # Изменить на нужное число
```

## Сравнение с search_2gis.py

| Аспект | search_2gis.py | search_outscraper.py |
|--------|---|---|
| Источник | 2GIS (локальный индекс) | Google Maps (через Outscraper) |
| Типы данных | Название, телефон, адрес | Название, сайт, рейтинг, отзывы |
| Скорость | Медленный (браузер Chrome) | Быстрый (API) |
| Точность | Высокая (локальные компании) | Высокая (глобальные) |
| Стоимость | Бесплатно | Платное (Outscraper) |
| Язык | Русский | Любой (параметр `language`) |

Выбирайте в зависимости от вашей задачи:
- Локальные компании в России → `search_2gis.py`
- Международные компании / сайты → `search_outscraper.py`
