# search_outscraper.py

Простой скрипт для поиска компаний через Outscraper (Google Maps API).

## Установка зависимостей

```bash
pip install outscraper python-dotenv openai
```

## Настройка .env

1. Скопировать `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Заполнить ключи:
   - `OUTSCRAPER_API_KEY` — ключ от Outscraper (https://outscraper.com)
   - `OPENAI_API_KEY` — ключ от OpenAI (https://platform.openai.com)
   - `OPENAI_MODEL` — модель (по умолчанию `gpt-4o-mini`)

## Использование

### 1. Создать query.txt

```bash
echo "тату-салоны в братске" > query.txt
```

Или отредактировать существующий файл.

### 2. Запустить скрипт

```bash
python search_outscraper.py
```

### 3. Результаты

Результаты сохранятся в `results.csv`:

```csv
name,site
ООО "Салон красоты",http://salon.ru
Тату-мастерская "Ink",https://inktattoo.com
```

## Логирование

Скрипт выводит прогресс:

```
[✓] Прочитан query.txt: 'тату-салоны в братске'
[✓] LLM распарсил: Ниша = 'тату-салоны', Город = 'братске'
[✓] Запрос к Outscraper: 'тату-салоны братске'
[✓] Найдено 42 компаний, из них 28 с сайтом
[✓] Результаты сохранены в results.csv
```

## Ошибки

Скрипт завершается с ошибкой и `exit(1)` если:
- Нет файла `query.txt`
- Нет переменных окружения в `.env`
- Ошибка API Outscraper или OpenAI
- Другие критические ошибки

## Структура query.txt

Формат произвольный, например:
- `кофейни в москве`
- `парикмахерские в санкт-петербурге`
- `врачи-кардиологи`

LLM автоматически распарсит нишу и город.

## CSV результаты

Колонки:
- `name` — название компании
- `site` — веб-сайт (если доступен)

Если сайта нет, колонка будет пустой.

## Максимальное количество результатов

По умолчанию ограничение в 100 компаний (параметр `limit=100` в скрипте).

Чтобы изменить, отредактировать строку:
```python
limit=100  # Изменить на нужное число
```
