🔗 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

ПУБЛИЧНЫЕ (без аутентификации):
   GET    /api/health              - Проверка статуса сервера
   GET    /api/proxies             - Получить доступные прокси
   GET    /api/proxies/<id>        - Получить прокси по ID
   POST   /api/vpn/toggle          - Включить/выключить VPN
   POST   /api/vpn/connect         - Подключиться к VPN
   POST   /api/vpn/location        - Изменить локацию
   GET    /api/speed-test          - Тест скорости

АДМИН (требует JWT токен):
   POST   /api/admin/login                  - Вход админа
   GET    /api/admin/verify-token           - Проверка токена
   GET    /api/admin/proxies                - Все прокси (с деталями)
   POST   /api/admin/proxies                - Создать прокси
   PUT    /api/admin/proxies/<id>           - Обновить прокси
   DELETE /api/admin/proxies/<id>           - Удалить прокси
   GET    /api/admin/users                  - Все пользователи
   PUT    /api/admin/users/<id>             - Обновить пользователя
   GET    /api/admin/statistics             - Статистика системы
   GET    /api/admin/settings               - Параметры системы
   POST   /api/admin/settings               - Сохранить параметры
   GET    /api/admin/logs                   - Системные логи
   DELETE /api/admin/logs                   - Очистить логи

🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ)
═══════════════════════════════════════════════════════════════════════════════

1. УСТАНОВКА BACKEND:
   cd ROOCOD/backend
   python3 -m venv venv
   source venv/bin/activate  (или venv\Scripts\activate на Windows)
   pip install -r requirements.txt
   python server.py

2. УСТАНОВКА РАСШИРЕНИЯ:
   Chrome/Edge:  chrome://extensions → Load unpacked → ROOCOD/extension
   Firefox:      about:debugging → Load Temporary Add-on → manifest.json

3. ИСПОЛЬЗУЙТЕ:
   • Нажмите на иконку расширения
   • Выберите локацию
   • Включите VPN
   • Для админа: Нажмите кнопку "Админ-панель"
   • Логин: admin / Пароль: admin123

🔐 БЕЗОПАСНОСТЬ
═══════════════════════════════════════════════════════════════════════════════

✅ Реализовано:
   • JWT аутентификация (JSON Web Tokens)
   • SHA256 хеширование паролей
   • CORS контроль доступа
   • Input validation и санитизация
   • HTTPS/TLS поддержка
   • Системное логирование всех действий

⚠️ DEVELOPMENT учётные данные:
   Admin Username: admin
   Admin Password: admin123
   JWT Secret: your-secret-key-change-in-production

🔒 PRODUCTION требования:
   • Измените все пароли
   • Генерируйте надёжный JWT_SECRET_KEY (32+ символов)
   • Используйте HTTPS с SSL сертификатом
   • Настройте правильные CORS origins
   • Включите логирование
   • Настройте резервное копирование БД
