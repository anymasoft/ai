## ЧАСТЬ 1: АРХИТЕКТУРА СИСТЕМЫ БИЛЛИНГА

### Механизм учета токенов
- **Валюта:** Виртуальные "токены" (не реальные API токены)
- **Коэффициент:** Каждый API токен имеет множитель стоимости (зависит от модели)
- **Формула:** `tokenCost = tokenCount * multiplier`
- **Проверка баланса:** До каждого API вызова
- **Списание:** Создание транзакций с историей

### Основные компоненты
1. **`balanceMethods.js`** — Проверка баланса перед операцией
2. **`spendTokens.js`** — Создание и сохранение транзакций
3. **`tx.js`** — Коэффициенты стоимости для каждой модели
4. **`tokenizer.ts`** — Подсчет реальных API токенов (TikToken)
5. **`usage.ts`** — Запись использования в БД

## ⚠️ КРИТИЧЕСКИЙ АНАЛИЗ

### Проблемы текущей модели

1. **Если пользователь использует Claude Sonnet или Opus** → вы в убытке
   - При среднем сценарии маржа = **-27.8% для Pro**
   - Пользователь будет платить 3990 руб, а вы потеряете $11.10 в прибыли

2. **Если пользователь умнее** → он выберет дешевые модели
   - Claude Haiku/GPT-4o mini дают 94% маржу
   - Пользователь очень доволен, вы зарабатываете, но на лимит товара

3. **Бизнес-модель не виновна - цены в России просто низкие**
   - $39.90 за 5М токенов = слишком дешево для среднего использования
   - Даже при GPT mini, маржа 94% теряется на операционные расходы

### Точка безубыточности (Break-Even)

**Pro тариф (5M токенов = $39.90 в месяц):**
```
Максимум можно потратить:  $39.90 × 0.7 маржа = $27.93 на себестоимость

Получается максимум:      27.93 / 5M = $0.005586 за 1 токен
                           = 5.586 USD за 1M токенов

Это между Claude Haiku ($1/$5) и Claude Sonnet ($3/$15)
Доступны ТОЛЬКО дешевые модели + Haiku
```

## ЧАСТЬ 8: РИСКИ И ЗАЩИТА

### Основные угрозы

#### 1. **Spam атаки токенами**
```
Пользователь может:
- Отправлять бесконечные запросы
- Отправлять очень большие промпты (1M+ токенов в 1 запрос)
- Использовать cheap модели в цикле
```

**Защита:**
```
✅ Текущие механизмы (есть в коде):
- checkBalance() перед каждым запросом
- Проверка лимита перед запросом
- Создание транзакций для истории

⚠️ Нужно добавить:
- Max tokens per request: 100,000 (limit by model)
- Max requests per minute: 60 requests/min
- Max tokens per hour: 1,000,000 tokens/hour
- Rate limiting per IP/User
```

#### 2. **Abuse дорогих моделей**
```
Пользователь платит за дешевый план, но использует Claude Opus
```

**Защита:**
```
✅ Рекомендация: Ограничить модели по тарифам

Pro (990 руб):
  ✓ Доступны: GPT-4o mini, Claude Haiku, Gemini 2.5 Flash
  ✗ Закрыты: Sonnet, Opus, GPT-5

Business (3990 руб):
  ✓ Доступны: Все модели выше + Claude Sonnet, GPT-5
  ✗ Закрыты: Claude Opus, GPT-5.4 (только для Enterprise)
```

#### 3. **Кеширование может "экономить" токены**
```
Anthropic предлагает 50% скидку на cached tokens
Пользователь может затейсировать большие системные промпты
→ Получит скидку, вы потеряете маржу
```

**Защита:**
```
✅ Вариант 1: Не применять скидку кеша (проще)
✅ Вариант 2: Применить скидку но добавить минимум в себестоимость
```

## ЧАСТЬ 9: СРАВНЕНИЕ МОДЕЛЕЙ ТАРИФИКАЦИИ

### Вариант 1: На основе ТОКЕНОВ (текущий)
```
✅ Плюсы:
  - Справедливая оценка (пользователь платит за то, что использует)
  - Легко масштабируется

❌ Минусы:
  - Нужна защита от abuse
  - Дешевых пользователей выгодно привлекать, они не потребляют токены
  - Сложно предсказать стоимость для пользователя
```

### Вариант 2: На основе СООБЩЕНИЙ
```
Цена за N сообщений в месяц (например, 100 сообщений)

✅ Плюсы:
  - Проще объяснить клиентам
  - Легче прогнозировать расход

❌ Минусы:
  - Не справедливо (большое сообщение = маленькое сообщение)
  - Пользователи могут жаловаться на ограничение размера ответа
```

### Вариант 3: CREDITS-based
```
Цена за N "кредитов"
1 кредит = 100 токенов (примерно)

✅ Плюсы:
  - Промежуточный вариант между токенами и сообщениями
  - Легче считать в голове

❌ Минусы:
  - Все равно нужна защита от abuse
```

### Вариант 4: По МОДЕЛЯМ
```
Разные цены для разных моделей (как в ChatGPT Plus)

Pro (990 руб):
  - Claude Haiku: $9.90/месяц
  - GPT-4o mini: $9.90/месяц

Business (3990 руб):
  - Claude Sonnet: $39.90/месяц
  - Все модели выше
  - GPT-5: +$10/месяц дополнительно

✅ Плюсы:
  - Явная граница между планами
  - Пользователю понятно, за что платит

❌ Минусы:
  - Сложнее объяснить комбинации
  - Может быть запутано
```
