## Классификация настроек

### A) USER SETTINGS (42 настройки) ✅
Безопасные настройки для пользователя:
- Theme, Language, Font Size
- UI опции (enterToSend, autoScroll, etc.)
- Speech параметры (STT/TTS)
- Аккаунт опции (username, avatar, 2FA, backup codes)

**Рекомендация:** Оставить у пользователя

### B) ADMIN SETTINGS (4 параметра) 🔴
Административные параметры, влияющие на стоимость:
- autoRefillEnabled
- refillAmount
- refillIntervalUnit
- refillIntervalValue

**Рекомендация:** Перенести в админку

### C) DANGEROUS OPERATIONS (5 операций) 🔴
Требуют усиленной защиты (2FA, логирование):
- DeleteAccount
- ClearChats
- RevokeKeys
- ImportConversations
- AgentApiKeys

**Рекомендация:** Требуют 2FA и логирования

## План внедрения

### Фаза 1: КРИТИЧНЫЕ (2-3 дня)
Начните НЕМЕДЛЕННО
```
- Добавить 2FA в DeleteAccount
- Добавить 2FA в ClearChats
- Добавить 2FA в RevokeKeys
- Валидировать ImportConversations
- Маскировать AgentApiKeys
```

### Фаза 2: ВАЖНЫЕ (5-7 дней)
Начните через неделю
```
- Валидировать Avatar загрузку
- Требовать 2FA для просмотра ключей
- Warning для cloudBrowserVoices
- Логирование критичных операций
- Email уведомления
```

### Фаза 3: УЛУЧШЕНИЯ (7-10 дней)
Начните через 2 недели
```
- Grace period для DeleteAccount (24ч)
- Бэкап для ClearChats
- Отложенный отзыв для RevokeKeys
- Синхронизация между устройствами
```

### Фаза 4: ОПТИМИЗАЦИЯ (10-15 дней)
Начните через месяц
```
- Ротация API ключей
- Admin audit log
- Алерты при подозрительной активности
- Документация пользователей
```
