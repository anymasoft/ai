# UI Документация — YouTubeAnalitycs (Beem Analytics)

## Общая структура приложения

После авторизации пользователь видит **dashboard layout** с:
- **Sidebar** слева (или справа — настраивается) с навигацией
- **Header** сверху с информацией о пользователе и настройками
- **Footer** снизу
- **Main Content Area** в центре с контентом каждой страницы

### Цветовая схема
- **Primary Color**: Синий (используется для кнопок, активных элементов)
- **Secondary Color**: Серый оттенок (фон карточек, muted элементы)
- **Background**: Светлый (white/light gray)
- **Text**: Тёмный (near-black для основного текста)
- **Borders**: Светло-серые линии

---

## SIDEBAR (Боковая панель)

### Структура
- **Высота**: Полная высота экрана (переносится вместе со скроллом)
- **Ширина**: 16rem (256px) в развёрнутом состоянии, 3rem (48px) в свёрнутом
- **Расположение**: По умолчанию слева, настраивается в Настройках

### Компоненты Sidebar

#### 1. **Header Logo**
```
┌─────────────────────────┐
│ [ЛОГОТИП] Beem Analytics│
│           Панель анали. │
└─────────────────────────┘
```
- Иконка приложения в синем квадрате (основной цвет, белый текст)
- Название "Beem Analytics"
- Подзаголовок "Панель аналитики"
- При клике переходит на /trending

#### 2. **Create Script Button**
```
┌─────────────────────────┐
│ ✨ + Создать сценарий   │
└─────────────────────────┘
```
- Full-width кнопка с иконкой Sparkles
- Primary color (синяя)
- При клике открывает /trending (генератор сценариев)

#### 3. **Navigation Groups**

**АНАЛИТИКА**
- 📊 **Обзор** → `/dashboard` - главная страница с ключевыми показателями
- 🎯 **Конкуренты** → `/competitors` - список добавленных конкурентов
- 🔄 **Сравнение** → `/competitors/compare` - сравнительный анализ конкурентов
- 📄 **История сценариев** → `/scripts` - список всех созданных сценариев
- ❓ **Вопросы и ответы** → `/faqs` - часто задаваемые вопросы
- 💬 **Обратная связь** → `/feedback` - форма обратной связи

**НАСТРОЙКИ**
- ⚙️ **Настройки** (с подменю)
  - 💳 **Биллинг** → `/settings/billing` - управление планами и платежами

#### 4. **Admin Section** (видна только администратору)
```
Администрирование
─────────────────
📬 Сообщения
```
- Видна только пользователю с правами администратора
- Кнопка для просмотра сообщений от пользователей

#### 5. **User Profile Footer**
```
┌─────────────────────────┐
│ [АВАТАР] Имя            │
│          email@user.com │
│ [Выход]                 │
└─────────────────────────┘
```
- Аватар пользователя
- Имя и адрес электронной почты
- Кнопка "Выход"

---

## HEADER (Верхняя панель)

### Структура
- **Высота**: примерно 56px
- **Background**: Светлый (muted фон)
- **Расположение**: Прилипает сверху содержимого

### Компоненты Header
- **Burger menu** (если sidebar свёрнут) - для открытия sidebar
- **Заголовок текущей страницы** (может отличаться для каждой страницы)
- **Правая часть**: Поиск, уведомления, профиль

---

## FOOTER (Нижняя панель)

### Структура
- **Background**: Светлый (muted)
- **Текст**: Маленький размер (text-sm, muted-foreground)
- **Контент**: Информация об авторе/компании и ссылки

## RESPONSIVE DESIGN

### Breakpoints (Tailwind CSS)
- **Mobile**: < 640px - Full width, stacked layout
- **Tablet**: 640px - 1024px - 2-column grid
- **Desktop**: > 1024px - 3-4 column grid, full sidebar

### Container Queries
- `@container/main` - Основной контейнер для адаптивной раскладки

### Sidebar Behavior
- **Desktop**: Всегда видна (или свёрнута в иконки)
- **Tablet**: Может быть скрыта/показана
- **Mobile**: Скрыта по умолчанию, открывается через бургер-меню

### Primary Colors
- **Primary**: #3b82f6 (Blue)
- **Primary Foreground**: #ffffff (White)

## TYPOGRAPHY

### Font Family
- **Sans**: Inter, -apple-system, BlinkMacSystemFont (Tailwind default)

### Font Sizes
- **Display**: 2xl - 3xl (для заголовков страниц)
- **Heading**: lg - xl (для заголовков секций)
- **Body**: base (16px) - для основного текста
- **Small**: sm (14px) - для подзаголовков, справки
- **Tiny**: xs (12px) - для мета-информации

### Font Weights
- **Regular**: 400
- **Medium**: 500 (для меню, ярлыков)
- **Semibold**: 600 (для подзаголовков)
- **Bold**: 700 (для заголовков)

## SPACING

### Margins & Padding (Tailwind Scale)
- **xs**: 0.25rem (4px)
- **sm**: 0.5rem (8px)
- **md**: 1rem (16px)
- **lg**: 1.5rem (24px)
- **xl**: 2rem (32px)

### Gaps (между элементами)
- Card items: gap-2 to gap-4
- Sections: gap-6 to gap-8

## SHADOWS & BORDERS

### Borders
- Default: 1px solid hsl(var(--border))
- Radius: sm (4px), md (6px), lg (8px), xl (12px)

### Shadows
- None (default)
- sm: Small shadow (hover states)
- md: Medium shadow (cards)
- lg: Large shadow (modals)

## ANIMATIONS & INTERACTIONS

### Hover States
- Buttons: Более тёмный оттенок, небольшое масштабирование
- Cards: Появляется тень, bg-muted/50
- Links: Появляется подчёркивание, основной цвет текста

### Focus States
- Inputs: ring-2 ring-primary, border-primary
- Buttons: ring-2 ring-offset-2 ring-primary

### Loading
- Spinner animations (animate-spin)
- Skeleton loaders (pulse animation)

### Transitions
- Duration: 200-300ms
- Easing: ease-in-out

## ACCESSIBILITY

- All buttons имеют видимые focus states
- Text контрастность соответствует WCAG AA стандартам
- Icons имеют aria-labels или являются декоративными
- Form inputs имеют associated labels
- Error messages объявляются для screen readers
- Keyboard navigation поддерживается везде
