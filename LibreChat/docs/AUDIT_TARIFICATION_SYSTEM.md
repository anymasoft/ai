### Структура тарифов (модель Plan)

Система использует **3 основных тарифа**, хранящихся в MongoDB (файл: `api/models/Plan.js`):

#### 1.1 ПЛАН "FREE" (Бесплатный)

```javascript
planId:                  'free'
label:                   'Free'
priceRub:                0 ₽
tokenCreditsOnPurchase:  0 (НЕ начисляется)
durationDays:            null (бесконечно)
allowedModels:           ['gpt-4o-mini', 'gpt-3.5-turbo']
```

**Важно:** Free план НЕ выдаёт токены! Пользователь получает начальные 25 сообщений от платформы при регистрации.

## 📊 РАЗДЕЛ 3: МЕХАНИЗМ СПИСАНИЯ ТОКЕНОВ

### 3.1 Архитектура списания

```
Request → checkBalance() → getMultiplier() → calculateTokenValue() → updateBalance()
                          ↓
                      tokenValues[model]
                      {prompt: X, completion: Y}
```

### 3.2 Формула расчета стоимости

```javascript
// Для каждого сообщения:

tokenCost = (promptTokens * multiplier_prompt) + (completionTokens * multiplier_completion)

где multiplier = tokenValues[modelId][tokenType]

Например для GPT-4o:
  prompt:     2.5 (за 1М input токенов)
  completion: 10  (за 1М output токенов)

Для сообщения с 100 prompt и 150 completion токенов:
  tokenCost = (100 * 2.5) + (150 * 10) = 250 + 1500 = 1750 credit-units
```

### 3.3 Механизм Auto-Refill

Если баланс упадёт ниже нуля, система может автоматически пополнить (если включено в настройках).

### 3.4 Файлы ответственные за списание

- `api/models/spendTokens.js` - логика создания транзакций
- `api/models/Transaction.js` - сохранение в БД и обновление баланса
- `api/models/balanceMethods.js` - проверка перед списанием
- `api/models/tx.js` - таблица цен (tokenValues)

## ⚠️ РАЗДЕЛ 7: АНАЛИЗ РИСКОВ

### 7.1 КРИТИЧЕСКАЯ УЯЗВИМОСТЬ

**Обнаружена несогласованность в системе:**

1. **Токены в Plan.js** = credit units (5M за Pro, 12M за Business)
2. **Цены в tx.js** = ₽ за миллион токенов API

```
Это РАЗНЫЕ единицы!

5,000,000 credit units ≠ 5,000,000 API токенов!

Если 1 credit unit ≠ 1 API токен, то цены могут быть
завышены или занижены в 50x раз!
```
