## 🔍 ОСНОВНЫЕ ВЫВОДЫ

### ✅ Что найдено:
1. **Точная причина:** getOverviewStats БЕЗ фильтра времени, getConversationStats С фильтром
2. **Где находится:** analyticsService.js строки 172-204 (Overview) vs 553-617 (Conversations)
3. **Почему расхождение:** Разные коллекции + разные фильтры = разные числа

### 🎯 Решение:
1. **Быстрое:** Добавить фильтр в getOverviewStats (30 мин)
2. **Правильное:** Унифицировать на Message коллекцию (2-3 часа)  
3. **Отладка:** Добавить логирование (15 мин)
