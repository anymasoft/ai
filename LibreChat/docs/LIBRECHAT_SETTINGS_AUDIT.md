#### Анализ по DATA

**Критические проблемы:**

1. **ImportConversations** 
   - Риск XSS при парсинге JSON с пользовательским контентом
   - Нужна валидация и санитизация
   - Может восстановить удалённые разговоры

   - Требует специального разрешения `REMOTE_AGENTS`

   - Требует двойного подтверждения
   - Может быть несчастным случаем

- **ImportConversations** - требует админского контроля
- **AgentApiKeys** - опасна, требует дополнительной защиты
- **RevokeKeys** - требует 2FA
- **ClearChats** - требует 2FA

**Рекомендация:** Перенести критичные операции в отдельный раздел с дополнительной защитой.

- Все из GENERAL (6)
- Все из CHAT (11) + FontSize
- Все из COMMANDS (3)
- Все из SPEECH (17)
- Все из PERSONALIZATION (1)
- Некоторые из DATA (SharedLinks - просмотр, DeleteCache)
- Некоторые из ACCOUNT (DisplayUsername, Avatar, 2FA, BackupCodes)
- Баланс (tokenCredits, lastRefill - read-only)

**Характеристики:**
- Локально сохраняемы
- Не требуют аудита
- Могут быть сброшены без проблем

---

### B) ADMIN SETTINGS (Административные)

Должны быть в отдельном разделе администратора, недоступны обычным пользователям.

**Список (3 настройки):**
- Balance auto-refill (autoRefillEnabled, refillAmount, refillIntervalUnit, refillIntervalValue)

**Характеристики:**
- Влияют на стоимость системы
- Требуют аудита
- Глобальные настройки

### C) DANGEROUS SETTINGS (Опасные)

Требуют дополнительной защиты, двойного подтверждения, 2FA, логирования.

**Список (5 критичных операций):**

1. **DeleteAccount** (ACCOUNT)
   - ✅ Email подтверждение
   - ⚠️ Требует 2FA
   - 🔴 Необратимо
   - Логирование: ОБЯЗАТЕЛЬНо

2. **ClearChats** (DATA)
   - ⚠️ Требует 2FA
   - 🔴 Необратимо
   - 📝 Двойное подтверждение: ОБЯЗАТЕЛЬНо
   - Логирование: ОБЯЗАТЕЛЬНо

3. **RevokeKeys** (DATA)
   - ⚠️ Требует 2FA
   - 🔴 Необратимо
   - Логирование: ОБЯЗАТЕЛЬНо
   - Может заблокировать интеграции

4. **ImportConversations** (DATA)
   - 🔴 XSS риск при парсинге
   - ⚠️ Требует администраторского одобрения
   - Валидация: ОБЯЗАТЕЛЬНа
   - Логирование: ОБЯЗАТЕЛЬНо

5. **AgentApiKeys** (DATA)
   - 🔴 КРИТИЧНАЯ операция
   - ⚠️ Требует специального разрешения
   - ⚠️ Требует 2FA для просмотра
   - Логирование: ОБЯЗАТЕЛЬНо
   - Хранение: Только на сервере, не в localStorage

**Характеристики:**
- Необратимые действия
- Требуют 2FA верификации
- Требуют явного согласия
- Требуют логирования
- Должны быть в отдельном разделе
